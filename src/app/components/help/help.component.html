{{ helpTitle }} La cola virtual es un sistema de generación de "tickets" virtuales que se genera cuando un cliente solicita una mesa y se le signa en el mismo momento que es solicitada. Cada ticket está numerado y en caso de que tu negocio esté lleno, servirá para administrar los "turnos" de ingreso a tu comercio. Sistema de Provisión de Mesas. Este sistema opera cuando la Cola Virtual esta activada, y si el SPM está activado el sistema asignará y proveerá automáticamente las mesas disponibles a los clientes que esperan una mesa en la cola virtual. Si esta opción NO esta activada el cliente debe esperar a que un camarero o administrador asigne su mesa. Agenda para reservas a futuro. Si ésta opción está activada, tus clientes podrán auto reservarse mesas disponibles para un día y horario deseado. El comercio debe tener definidos los horarios y días hábiles. El administrador también puede hacer reservas o re-asignar mesas ya reservadas de forma manual.
Si el cliente solicita una mesa para un grupo de personas para el cual no existe mesa que pueda cumplimentar el requerimiento, tendrá la opción de iniciar una solicitud de mesa especial. Cuando la hora de reserva se encuentre próxima, el sistema comenzará a reservar las mesas asignadas en forma automática y a la hora de la reserva proveerá la mesa (o las mesas) y notificará al cliente. Cantidad máxima de intervalos de 30 minutos que un cliente puede tomar en una reserva. Los intervalos deben ser consecutivos. Este límite no aplica al momento de crear un ticket de contingencia desde la agenda.
